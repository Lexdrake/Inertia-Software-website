How do I download the beta?

+

iPhone users: Download TestFlight from the App Store, then use the beta code provided in your invitation.

Android users: Use the Google Play Console link and beta code from your invitation.

If you need the codes again, just email us!

I'm new to the app - where should I start?

+

Check out the Getting Started Guide for a complete walkthrough!

Quick start: Pick one mode that fits your current situation (Student, Work, or Life), create a few items, then use the Planner tab to schedule them. Don't try to set up everything at once - start small and build from there.

Do I have to use all three modes?

+

Nope! Each mode is completely self-sufficient. You can:

  • Use just Student Mode if you're focused on academics
  • Use just Work Mode for professional planning
  • Use just Life Mode for personal goals
  • Mix and match any combination that works for you

The app is designed to be useful even if you only use one mode.

üì± Using the App

How does batch scheduling work?

+

Batch scheduling lets you schedule multiple items at once:

  1. Go to the Planner tab ‚Üí All Items
  2. Select the items you want to schedule (checkboxes)
  3. Tap the scheduling button to move to weekly view
  4. Click an item from your selected list, then click a green time slot
  5. Repeat until everything is scheduled

Green slots mean that time is available for the selected item type.

What's the difference between projects in different modes?

+

Projects work the same way (milestones and tasks) but have different contexts:

  • Student projects: Assigned to a specific class (like "Research Paper for History 101")
  • Work projects: Professional deliverables (like "Q1 Marketing Campaign")
  • Life goals: Personal projects nested within broader goals (like "Learn Guitar" within a "Creative Growth" goal)

How do habits work?

+

Habits are time-based goals you can set to:

  • Maintain: Keep doing something at your current level
  • Increase: Gradually do more of something good
  • Decrease: Gradually do less of something you want to reduce

You can report on habits in the Productivity tab, or set them as "omitted" if you prefer not to track them actively.

What are Work Mode expectations?

+

Expectations are goals you can set at different time scales that cascade down:

  • Annual: Big yearly goals
  • Quarterly: 3-month milestones toward annual goals
  • Monthly: Steps toward quarterly goals
  • Weekly/Daily: Specific actions to hit monthly goals

You can set check-in schedules to regularly update your progress on each expectation.

The app crashed - what should I do?

+

Sorry about that! Please help me fix it:

  1. Try reopening the app
  2. If it keeps crashing, try restarting your phone
  3. Submit a bug report with details about what you were doing when it crashed
  4. Include your device model and iOS/Android version

You can also email us directly - crashes are high priority to fix!

+

This could be either a bug or a design that needs improvement. Either way, I want to know!

Please submit feedback describing:

  • What you were trying to do
  • What actually happened
  • What you expected to happen

This kind of feedback is super valuable for making the app more intuitive.

I can't find something that should be there

+

A few possibilities:

  • It might be in a different tab than you expect
  • It might be a feature coming in Beta v2.0 (check the timeline)
  • It might be a UI issue where something isn't visible when it should be

Feel free to ask us directly - we're happy to help you find it or confirm if it's not implemented yet!

üéØ About the App

What's the "inertia" concept about?

+

It's based on the physics principle: "Objects at rest stay at rest, objects in motion stay in motion."

The app is designed to help you:

  • Get in motion: Break through procrastination and resistance with clear, actionable plans
  • Stay in motion: Maintain momentum by always knowing what you can do next
  • Build momentum: Create systems that make it easier to keep going

Instead of fighting against inertia, the app helps you work with it.

The timeline depends on how the beta testing goes! Here's the rough plan:

  1. Beta v1.0: ~3 weeks (current)
  2. Beta v2.0: Advanced features after v1.0 is solid
