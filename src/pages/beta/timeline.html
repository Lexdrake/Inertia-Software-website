Inertia Planner Development Timeline

From beta testing to public launch - here's the complete roadmap

📍 Where We Are Now

Preparing for Beta v1.0 launch with core functionality across all three life modes. You're here at the perfect time to help shape the app's development!

🚀 Beta v1.0 - Foundation (Launching Soon)

Duration: ~3 weeks Launching Soon

The core Inertia Planner experience with essential features for managing your student life, work, and personal goals. This version focuses on the fundamental planning and scheduling functionality.

🎓 Student Mode

Core Features
  • ✅ Create and manage classes
  • ✅ Add assignments and test prep items to classes
  • ✅ Academic project management with milestones and tasks
  • ✅ Academic habit tracking (time-based goals)
Available Tabs: School • Habits • Home • Planner

💼 Work Mode

Core Features
  • ✅ Professional expectation tracking (deliverable & metric-based)
  • ✅ Cascading expectations (annual → quarterly → monthly → weekly → daily)
  • ✅ Work project management with milestones and tasks
  • ✅ Work routine management (recurring scheduled items)
Available Tabs: Work • Routines • Home • Planner

🌟 Life Mode

Core Features
  • ✅ Personal goal setting with nested projects and habits
  • ✅ Actionable goal breakdown into manageable tasks
  • ✅ Life habit tracking integrated with goals
Available Tabs: Goals • Home • Planner

📅 Central Planner (All Modes)

Planning & Scheduling
  • ✅ All Items tab - view everything across modes
  • ✅ Batch scheduling - select multiple items to schedule
  • ✅ Weekly view scheduling with green time slot indicators
  • ✅ 3-day planner view with item management
  • ✅ Productivity tab for progress reporting
  • ✅ Task/assignment completion tracking
  • ✅ Habit reporting (with omit option)
🔄 Expected Sub-Versions

During the 3-week beta period, you may see updates like v1.1, v1.2, v1.3, etc. These will focus on:

  • 🐛 Bug fixes based on your reports
  • 🎨 UI/UX improvements from your feedback
  • ⚡ Performance optimizations
  • 🔧 Minor feature tweaks and refinements

The final version number (v1.X) depends on what we discover together during testing!

🔮 Beta v2.0 - Advanced Features

Duration: TBD after v1.0 Upcoming

Building on the solid foundation of v1.0, this phase introduces collaboration features, advanced life management tools, and social connectivity.

👥 Collaboration Features

Student Groups
  • 🔄 Create groups between users
  • 🔄 Collaborative project creation within groups
  • 🔄 Shared task management and progress tracking
Work Teams
  • 🔄 Team creation with cascading expectations architecture
  • 🔄 Forum-style messaging system
  • 🔄 Nested topic replies (up to 4 levels deep)
  • 🔄 Long message length support
New Tabs: Groups (Student) • Teams (Work)

🌟 Life Mode Expansion

Smart Goal Creation
  • 🔄 AI-powered goal creation assistance
  • 🔄 Automatic project and habit generation based on goals
  • 🔄 Personalized recommendations and strategies
Event Management
  • 🔄 Holiday and important date calendar
  • 🔄 Vacation planning functionality
  • 🔄 Check-in system for memories and notes
  • 🔄 Option to clear planner items during vacations
New Tab: Events (calendar view and vacation planning)

🌐 Social Features

Friends & Social Feed
  • 🔄 Add friends and manage friends list
  • 🔄 Social feed with post sharing
  • 🔄 Calendar sharing between friends
Group Calendars
  • 🔄 Create calendar groups with friends
  • 🔄 View shared free time blocks across group members
  • 🔄 Color-coded availability (green: full day, orange: 6+ hours, red: 2-6 hours)
Social Events
  • 🔄 Create and manage social events
  • 🔄 Invite groups and individual friends
  • 🔄 Open invite toggle (friends can invite friends)
  • 🔄 RSVP system with accept/decline/unsure options
  • 🔄 Host announcements and Q&A messaging
New Tab: Social (feed, group calendars, social events)
🔄 Expected Sub-Versions

Similar to v1.0, v2.0 will likely have multiple iterations (v2.1, v2.2, etc.) as we:

  • 🐛 Fix any issues with new collaboration features
  • 🎨 Refine the social and event interfaces
  • ⚡ Optimize performance with increased data complexity
  • 🔧 Perfect the smart goal creation system
